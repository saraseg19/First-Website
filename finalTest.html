The two main parts of an HTML document are the: <head> and <body> sections. The <head> section specifically holds the document information such as document title, coding standards, external style sheet links, and script code.

Forms <form>

Forms are surveys that users can fill out in a webpage. The form tag is the method of creating a form, but it is a container element that holds other elements.

Div Tag <div>

Div elements are containers for other HTML elements. The internal elements can be moved around the webpage or stylized a certain way, and this only affects the elements that are in the div tags.

Footer Tag <footer>

The footer section of a webpage can contain several links to other pages in the site. This is a second navigation section that can contain links to some pages that are not necessarily main content. Some examples include ‘Contact Us’, ‘Privacy’, or a several social media accounts.

Table Tag <table>

Tables are structures that websites can use to sort their content. The table tag is the container for the table row and table cell tags; however, the table row is another container, so the cell is where the contents are actually stored.

Paragraph Tag <p>

Paragraphs simply contain text, but the text can be stylized using CSS. Paragraphs themselves can also be stylized, such as with their width and height.

HTML5 Tags, CSS, and Links

The tags <article> and <figure> are some new semantic tags introduce in HTML5.

  • The <article> tag contains the main content of the page
  • The <figure> tag is similar to the <img> tag in that it contains an image, but it is actually a container tag for an image so it will still need the <img> tag

There are three levels of CSS: external, internal, and inline.

  • External - Uses a separate HTML page. This page is linked in the head section of an HTML file.
  • Internal - This way uses a <style> tag and is located in the <head> section of the page.
  • Inline - This form of CSS is located inside an element.

Unified Look and Feel

Sites should present themselves in a uniformed fashion. This gives the user a better looking experience with the site and it also helps them navigate the site much easily. Sites should have a navigation function included somewhere noticeable. Having this navigation section in the same place will make sure that users can go to any point in the site and not get lost.

Having uniformed site helps out web developers as well. Reusing content can allow sites to load pages much quicker, which makes users not want to leave. Developers can also reuse their page's template throughout the website.

When making sites for mobile users, some companies might want to create a separate site all together. It's still important for developers to try to make the two sites similiar to one another for users.

Box Properties

Boxes have several layers to them known as: padding (inside the border), border, and margin (outside the border). These properties can be manipulated using CSS. Padding and border usually have their sizes changed and borders can be present or not.

Borders have the most styling options out of the three properties. If they are present they can be dotted, solid, dashed, double, and many more styles. Borders can also change sizes, in fact each side of a border can have a different size value if that is what the developer wants!

Padding and margins can have multiple sizes to them as well. This ability is useful when managing a site's layout or creating cool font effects.
