Information About HTML

Definition List

Attribute
Code added to an element to provide more information about the element.
Browser
A computer program with a graphical user interface for displaying and navigating between web pages.
Directory
A file which consists solely of a set of other files.
Meta Data
Information about the document itself, such as how to present the document, or what other documents are related to the current one.
Uniform Resource Locator (URL)
The address of a web page.

Six xHTML Rules

  • Elements must nest symmetrically.
  • Elements names are case sensitive. (Convention is to use lowercase values.)
  • End tags are required.
  • Empty elements are signified with a closing slash.
  • Attribute values must be contained in quotation marks.
  • Attribute–value pairs must be written in full (e.g. checked:checked;).
